Installationsaleitung des Programmes

Disclaimer

Das Programm wurde für die Verwendung mit einem Raspberry Pi 3 oder 4 entwickelt. Es kann auch auf ähnlichen Geräten, Mac, Windows und anderen Linux Plattformen funktionieren.

In Bezug auf die Installation wird hier nur die vom Entwicklungsteam verwendete und verifizierte Vorgehensweise gezeigt. Der Abschnitt Quickstart beinhaltet alle nötigen Befehle in der richtigen Reihenfolge. Genaure Erklärungen befinden sich weiter unten.

Quickstart

Nur verwenden wenn du weißt was du tust. Nur für fortgeschrittene Nutzer empfohlen. Die Vorraussetzungen stehen weiter unten.

$ sudo apt-get update
$ sudo apt-get install cmake libsdl2-dev
$ git clone https://github.com/Raspi64/raspi64.git
$ ./build.sh

Vorraussetzungen

1. Hardware

Getestet wurde das Projekt mit einem Raspberry Pi 3B und Raspberry Pi 4. Es kann auch auf älteren und ähnlichen Modellen funktionieren allerdings übernehmen ist das nicht garantiert.

2. Betriebssystem

Raspberry Pi OS with Desktop Release 2 Dezember 2020 Es kann auch auf anderen Versionen funktionieren allerdings übernehmen ist das nicht garantiert.

3. Pakete

  • cmake
  • libsdl2-dev

Die Pakete können über folgende Befehle installiert werden:

$ sudo apt-get update
$ sudo apt-get install cmake libsdl2-dev

Eigentliche Installation des Programmes

Es wird davon ausgegangen das die Vorraussetzungen erfüllt werden.

1. Klonen des Projektes

Das Projekt wird mit allen benötigten Komponennten und älteren Versionen heruntergeladen und im aktuellen Ordner gespeichert gespeichert.

$ git clone https://github.com/Raspi64/raspi64.git

2. Build.sh

Das Programm wird nun kompiliert und ausführbar gemacht. Dies kann etwas dauern. Bitte drücke keine Tasten um Fehler zu verhindern. Die Warnings sind Teil des Vorganges und normal, du musst dir keine Sorgen machen deswegen.

$ ./build.sh

3. Starten des Programmes

Das Programm kann nun mit diesem Befehl gestartet werden. Alles zur Verwendung des Programmes kannst du dir im Hilfesystem ansehen.

$ ./imgui_setup
